# Scoop Bucket for Kleio

This is the official Scoop bucket for [Kleio CLI](https://github.com/kleio-build/kleio-cli).

## Installation

```powershell
# Add the bucket (one-time setup)
scoop bucket add kleio https://github.com/kleio-build/scoop-bucket

# Install
scoop install kleio
```

## Update

```powershell
scoop update kleio
```

## Uninstall

```powershell
scoop uninstall kleio
scoop bucket rm kleio  # optional
```

## Available Apps

| App | Description |
|-----|-------------|
| `kleio` | Kleio CLI — capture work discovered during development; includes `kleio mcp` for MCP |

## About Kleio

Kleio CLI captures work items and backlog signals during development. Learn more at [github.com/kleio-build/kleio-cli](https://github.com/kleio-build/kleio-cli).
